## Summary
This app lets users search the GiphyApi for gifs. They can click on images to tweek the gifs via Twitter Cards. This project requires NodeJs.

## Installation

* Clone repository
* cd into repository
* run "npm install"
* run "npm start"
* View live demo at my [Surge Page](http://giphy.surge.sh)
